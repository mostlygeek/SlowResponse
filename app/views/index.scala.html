About

This is a simple endpoint for testing HTTP async response code. This server will wait n milliseconds before returning a regular text response or javascript.

Text response - regular text/plain response.

  • timeout 0ms
  • timeout 10ms
  • timeout 100ms
  • timeout 1000ms
  • timeout 5000ms
  • timeout 10000ms

AJAX (JSONP) Response

Use these in a <script> tag. These are good for testing w/ a JSONP style request so they work across domains.

You can specify your own callback function as the second parameter. By default it is alert().

  • timeout.js 0ms
  • timeout.js 10ms
  • timeout.js 100ms
  • timeout.js 1000ms
  • timeout.js 5000ms
  • timeout.js 10000ms
